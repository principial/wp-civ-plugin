Plugin adds image and video from YouTube or Vimeo

== Description ==

Plugin adds image and video from YouTube or Vimeo to the taxonomy “Category”
Plugin shows them before description on archive page if your theme use category_description filter

== Installation ==

Upload the Custom category image and video plugin to your blog, Activate it.
You're done!
